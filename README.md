# React Memoization

Memorizationよるパフォーマンスへの影響と有効性について調べるためのリポジトリです。

計測には、performance APIを用いています。

*created by create-react-app*

## Quick start
### development
``` shell
yarn
yarn start
```
### production
Only the first time.
```shell
yarn global add serve
```

``` shell
yarn
yarn build
serve -s build
```
